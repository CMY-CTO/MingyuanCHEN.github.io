Projects

Intelligent Traffic Sign Recognition System

Sep 2025 – Dec 2025

Problem: This project aimed to develop an image classification system for traffic sign recognition, which can support intelligent transportation systems and autonomous driving applications. The goal was to improve the automated identification of road signs from visual input in order to enhance recognition efficiency and safety.

Data: The project used a public traffic sign image dataset containing thousands of labeled images across multiple traffic sign categories. The data included image files with different lighting conditions, angles, and background complexity, which made the task closer to real-world visual recognition scenarios.

Approach: I performed image preprocessing, resizing, normalization, and data augmentation before training classification models for traffic sign recognition. The workflow included exploratory analysis, feature extraction experiments, and the implementation of convolutional neural network models to improve classification performance.

Outcome / Impact: The final model achieved strong classification performance on validation data and demonstrated the feasibility of using deep learning for traffic sign recognition tasks. The project provided a complete computer vision pipeline from data preprocessing to model evaluation and highlighted the value of AI in intelligent transportation applications.

My Contribution: This was an individual project. I completed the full pipeline independently, including dataset preparation, preprocessing, model development, evaluation, result interpretation, and project documentation.

News Text Classification for Topic Detection

Jan 2026 – Apr 2026

Problem: This project focused on building an NLP-based text classification system to automatically categorize news articles by topic. The business and research value of the project lies in improving information organization, content filtering, and large-scale text analysis efficiency.

Data: The dataset consisted of labeled news articles from multiple topic categories such as politics, business, technology, and sports. The data included raw text content and corresponding labels, allowing supervised learning methods to be applied for topic prediction.

Approach: I cleaned and tokenized the text data, transformed textual information into machine-readable features using standard NLP techniques, and trained classification models for topic prediction. The project compared different text representation and machine learning approaches to identify an effective pipeline for document classification.

Outcome / Impact: The final model showed reliable topic classification performance and demonstrated how NLP methods can be applied to organize unstructured text efficiently. The project also provided practical experience in text preprocessing, feature engineering, and supervised learning for language data.

My Contribution: This was an individual project. I was responsible for the complete workflow, including data preprocessing, feature extraction, model training, evaluation, and final reporting.
